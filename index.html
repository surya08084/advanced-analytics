Prerequisites & Setup

What You Need to Know

Basic Python (lists, dicts, functions, classes) and a vague memory of high-school statistics. That's it. Everything else is built here from first principles.

Python Environment Setup

Every chapter uses this standard stack. Set it up once, use it everywhere.

# Create a clean environment
conda create -n analytics python=3.11
conda activate analytics

# Core stack — used across all chapters
pip install numpy pandas scipy scikit-learn matplotlib seaborn
pip install statsmodels plotly jupyter notebook ipykernel

# Chapter-specific (install when needed)
pip install xgboost lightgbm catboost          # Ch 3, 4
pip install prophet neuralprophet              # Ch 5
pip install umap-learn hdbscan                 # Ch 6
pip install pymc arviz                         # Ch 8
pip install networkx pyvis                     # Ch 9
pip install transformers datasets              # Ch 10
pip install mlflow evidently                   # Ch 11
pip install dbt-core sqlalchemy                # Ch 12
📌 Note on Datasets

Every code example uses either built-in datasets (sklearn, seaborn) or freely downloadable public datasets. No paid data sources. All examples are reproducible on a laptop with 8GB RAM.
